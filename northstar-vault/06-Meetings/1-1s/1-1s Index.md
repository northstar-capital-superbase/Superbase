# 1:1s

> [!info] Why this page exists
> 1:1s carry ongoing context about a specific working relationship — open threads, growth conversations, recurring concerns — that shouldn't be scattered across dated notes with no throughline. Each person gets one durable hub page.

> [!tip] When to use this page
> Create a hub page (from [[Recurring Meeting Template]], or a dedicated 1:1 variant if the team's needs diverge) the first time someone starts having regular 1:1s with a manager. Log each actual conversation as a dated note in `06-Meetings/Notes/` using [[1-1 Template]], linked back via the `with` frontmatter field.

> [!note] How it connects to the rest of the vault
> Links to the person's entry in [[People & Org]]. Sensitive personal content should stay high-level here if the vault is ever shared beyond the direct manager — treat 1:1 hub pages as manager-owned, not fully public by default once the org grows.
